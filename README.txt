Mikko Test by Armand Krijgsman
Required: Git, Composer, PHP 5.5+ CLI

To proceed please clone this Repository manie20/mikkotest from GitHub.
When the code is on your local/remote server

1. Create a working directory:

    # mkdir test
    # cd test

2. Clone the code from GitHub:

    # git clone https://github.com/manie20/mikkotest.git .

3. Run Composer:

    # composer install

3. Test the application:

    # ./salaries.php help

4. Generate the CSV file as test.csv

    #  ./salaries.php --name test.csv

That's it!
